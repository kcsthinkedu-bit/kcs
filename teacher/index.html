교사 제출함 · 반 코드 관리

이 페이지에서 학생 제출물을 확인하고, 실제로 사용할 반 코드를 생성·복사·비활성화·삭제할 수 있습니다. 학생이 같은 이름으로 다시 제출하면 기존 제출을 덮어써서 정리하기 쉽게 바꿨습니다.

홈으로

반 코드 생성

학급 이름은 필수이고 메모와 직접 코드는 선택입니다. 반 코드를 만들면 학생이 그 코드를 입력해 접속할 수 있습니다.

사용 안내

추천 흐름은 다음과 같습니다. ① 반 코드를 만든다 → ② 학생에게 코드를 알려준다 → ③ 학생이 반 코드와 이름을 입력해 작업한다 → ④ 제출함에서 반 코드로 검색해 모아서 본다.

활성 코드만 입장 가능
학생은 교사가 생성한 활성 코드를 입력해야 시작할 수 있습니다.
코드 복사 버튼 제공
학급 채팅방이나 클래스룸에 바로 붙여 넣기 쉽도록 복사 버튼을 넣었습니다.
비활성화 / 삭제 가능
수업이 끝난 반 코드는 비활성화하거나 완전히 삭제할 수 있습니다. 제출물도 필요한 경우 교사 페이지에서 삭제할 수 있습니다.
