컬러링북 편집기 MVP

학생/선생님이 같은 홈페이지 안에서 함께 사용하는 기본 편집기

현재 모드: 학생

페이지 목록

학생/선생님이 공통으로 보는 프로젝트 페이지

1 페이지

    미리보기

    학생과 선생님이 같은 결과물을 함께 확인하는 공통 영역

    아직 이미지가 없습니다.
    화면을 추가하고 이미지를 불러오면 여기에서 바로 확인할 수 있습니다.

    현재 선택 페이지: 없음

    모드 패널

    한 홈페이지 안에서 학생/선생님 기능을 분리해 표시

    학생 작업 패널

    학생은 기본 작업만 빠르게 사용합니다. 화면 추가, 이미지 불러오기, JSON 불러오기를 우선 제공합니다.

    현재 할 수 있는 작업
    화면 추가 · 이미지 불러오기 · JSON 불러오기 · 페이지 선택
    학생 모드에서는 복잡한 설정을 숨기고, 결과 확인과 기본 입력에 집중합니다.

    선생님 작업 패널

    선생님은 같은 페이지 안에서 조금 더 자세한 설정을 조정합니다. MVP에서는 최소한의 항목만 둡니다.

    이 패널은 같은 홈페이지 안에 존재하지만, 학생 패널과 로직을 분리해 관리하는 MVP 뼈대입니다.
