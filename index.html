KCS 책 만들기 학생 모드 선생님 모드

우리 반 책 만들기를
더 쉽게 시작하세요

이 시작 페이지는 실제 편집기 입구입니다. 학생은 글과 이미지를 쉽게 넣고, 선생님은 점검판에서 빠진 페이지를 바로 찾아 수정한 뒤 인쇄 배열까지 확인할 수 있습니다.

책 만들기 시작 샘플 JSON 내려받기 사용 순서 보기
• 실제 편집기는 /coloringbook/ 아래에서 동작합니다.
• 저장은 JSON, 불러오기는 현재 KCS 형식 파일 기준으로 작동합니다.
• 인쇄 전에는 선생님 모드의 점검판과 책 인쇄 배열을 함께 확인하세요.

지금 할 수 있는 핵심 작업

학생용 빠른 제작

표지 제목, 펼침 글, 이미지 넣기, Ctrl+V 붙여넣기, 저장과 다시 불러오기까지 바로 진행할 수 있습니다.

선생님용 점검

문제 페이지 찾기, 이미지 중앙 정렬, 전체 글자 크기 조정, 인쇄 전 상태 확인을 한 번에 처리합니다.

최종 인쇄 배열

책 페이지 순서에 맞는 시트 카드형 인쇄 배열을 확인한 뒤 브라우저 인쇄로 이어질 수 있습니다.

학

학생 모드

복잡한 조정값은 숨기고 글과 이미지를 넣는 기본 작업에 집중할 수 있게 구성했습니다.

교

선생님 모드

부족한 페이지 찾기, 전체 이미지 정리, 글자 크기 일괄 조정 등 마감 전 점검 기능을 제공합니다.

인

인쇄 배열 확인

긴 문서형이 아니라 시트 카드형으로 보이도록 구성해 실제 출력 전에 순서를 빠르게 확인할 수 있습니다.

추천 사용 순서

1

편집기 열기

먼저 편집기로 이동해 책 제목, 표지 제목, 펼침 내용을 입력합니다.

2

이미지와 글 채우기

파일 넣기나 Ctrl+V로 이미지를 채우고, 본문과 제목을 정리합니다.

3

JSON 저장

현재 작업본을 JSON으로 저장해 두면 같은 형식으로 다시 이어서 편집할 수 있습니다.

4

선생님 점검 후 인쇄

선생님 모드에서 빠진 페이지를 확인하고 인쇄 배열로 넘어가 최종 출력합니다.

빠른 바로가기

필요한 곳으로 바로 이동할 수 있게 주요 링크를 정리했습니다.

편집기 열기 샘플 JSON 주의사항 보기
처음 접속한 사용자가 "왜 바로 편집기가 안 보이지?" 하고 헷갈리지 않도록, 이제 루트 주소에서는 안내형 시작 화면이 먼저 보이고 실제 작업은 편집기에서 이어집니다.

이 시작 페이지에서 알 수 있는 것

  • 학생 모드와 선생님 모드가 어떻게 나뉘는지
  • 저장(JSON), 불러오기(JSON), 인쇄 배열 기능이 있다는 점
  • 실제 편집 화면은 /coloringbook/ 경로에 있다는 점

주의할 점

  • 예전 형식 JSON은 바로 안 열릴 수 있으므로 현재 형식으로 다시 저장해 두는 것이 안전합니다.
  • 인쇄 전에는 선생님 모드 점검판과 인쇄 배열을 함께 보는 것이 좋습니다.
  • 이미지 위치가 어긋나면 선생님 모드에서 중앙 정렬 또는 위치 초기화를 먼저 해 주세요.
